What is Hypertext Markup Language?

HTML is not a programming language but a markup language. It uses a markup system composed of elements which represent specific content. Markup means that with HTML you declare what is presented to a viewer, not how it is presented. Visual representations are defined by Cascading Style Sheets (CSS) and realized by browsers. however existing elements that allow for such exist. These elements are considered entirely obsolete and must not be used by authors.

Notes on some HTML elements

The following notes are based on DevDocs.io and Jon Duckett's HTML and CSS book:

Each element has an opening tag and a closing tag. Tags act like containers. They tell you something about the information that lies between their opening and closing tags.


Strong

Demonstrating the strong tag:
<strong></strong> Lorem ipsum dolor sit amet
The strong element indicates that the content within the tags have a STRONG importance. By default browsers will show the contents in bold.


Emphasis

Demonstrating the emphasis tag:
<em></em> Lorem ipsum dolor sit amet
This element indicates emphasis that subtly changes the meaning of a sentence. By default browsers will show the contents in italics.


Blockquote

Demonstrating the blockquote tag:
<blockquote></blockquote>
"Lorem ipsum dolor sit amet, consectetuer adipiscing elit. Aenean commodo ligula eget dolor."
This element is used for longer quotes. Browsers tend to ident blockquotes.


Quote

Demonstrating the quote tag:
<q></q> Lorem ipsum dolor sit amet, consectetuer adipiscing elit. Aenean commodo ligula eget dolor.
This element is used for shorter quotes but is generally avoided as Internet Explorer does not put quotation marks around the tag.


Citation

Demonstrating the citation tag:
<cite</cite> Lorem ipsum dolor sit amet, consectetuer adipiscing elit. Aenean commodo ligula eget dolor.
This tag is used to describe a reference to a cited creative work, and must include either the title or author or the URL of that work.


Address

Demonstrating the address tag:
<address></address> For example:
<address><p><a href="mailto:homer@example.org">homer@example.org</a></p> <p><742 Evergreen Terrace, Springfield</p></address> This tag is used to contain contact details of the author.


Inserted and Deleted

Demonstrating the insert and deletion tag:
<ins></ins> This tag is used to show content that has been inserted. <del></del> This tag is used to show text that has been deleted from it. It is usually presented as crossed out.
Moreover, the <s></s> element is used for something that is no longer accurate or relevant (but should not be deleted). For example:
<p><s>Was $995</s></p> <p>Now only $375</p>


Lists

There are three different types of lists:
  • Ordered lists
  • Unordered lists
  • Definition lists

The differences b/wn these lists are as follows:
  • Ordered lists are lists where each item in the list is numbered
  • Unordered lists are bullet pointed.
  • Definition lists are made up of terms along with the definitions for each of those terms.


Style

<style></style> This element contains style information for a document. It contians CSS (Cascading Style Sheets) which is applied to the contents of the document.
